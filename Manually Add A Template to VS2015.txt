To manually add a new Template to Visual Studio 2015, copy the zip file to the following location:

%USERPROFILE%\Documents\Visual Studio 2015\Templates\ProjectTemplates\Visual C#\Windows

For example, copy T10Minimal.zip to

c:\users\username\Documents\Visual Studio 2015\Templates\ProjectTemplates\Visual C#\Windows

You do not need to expand the zip file.
